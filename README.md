# 🛒 Unimarket-BUAP

Plataforma web tipo marketplace enfocada en la comunidad universitaria de la BUAP, donde los usuarios pueden publicar, explorar y gestionar productos de forma sencilla.


## 🧠 Problema

Dentro de la comunidad estudiantil no existe una plataforma centralizada para compra/venta de productos entre alumnos, lo que genera desconfianza y desorganización.

## 💡 Solución

Unimarket-BUAP permite:

* Publicar productos fácilmente
* Explorar artículos por categoría
* Gestionar publicaciones propias
* Interacción rápida entre usuarios

## 🧰 Tecnologías utilizadas

* ⚛️ React
* ⚡ Vite
* 🔥 Firebase (Auth + Firestore)
* 🎨 CSS / Tailwind 

## 🔐 Funcionalidades principales

* Autenticación de usuarios con Firebase
* CRUD de productos
* Base de datos en tiempo real
* Interfaz rápida y responsiva

## 📦 Instalación

```bash
npm install
npm run dev
```

## 📁 Estructura del proyecto

* `/src/components` → Componentes reutilizables
* `/src/pages` → Vistas principales
* `/src/firebase` → Configuración de Firebase
